# EV PATH: Electric Neon Design System

## 🎨 Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| **Primary** | `#00ff88` | Neon Green - Buttons, Accents, Highlights |
| **Secondary** | `#00d1ff` | Electric Blue - Secondary charts, info icons |
| **Background** | `#0a0a0a` | Deep Charcoal - Main app background |
| **Surface** | `rgba(255, 255, 255, 0.05)` | Glassmorphism - Cards, Modals |
| **Text Primary** | `#ffffff` | High contrast text |
| **Text Secondary** | `rgba(255, 255, 255, 0.7)` | Low priority text |

## Typography
- **UI & Body**: `Inter`, Sans-serif
- **Data & Numbers**: `Roboto Mono`, Monospace (for that "High-Tech" feel)
- **Headings**: `Outfit` or `Inter` (Bold)

## Visual Effects
- **Glassmorphism**: 
  - `background: rgba(255, 255, 255, 0.05)`
  - `backdrop-filter: blur(12px)`
  - `border: 1px solid rgba(255, 255, 255, 0.1)`
- **Glow**: 
  - `box-shadow: 0 0 15px rgba(0, 255, 136, 0.3)` (Used for Primary buttons)

## Components
### Achievement Badges
- **Shape**: Hexagonal or circular with a glowing border.
- **Icon**: Minimalist SVG in the center.

### Profile Garage
- **Grid**: 2x2 grid for statistics.
- **Card**: Glassmorphic cards with staggered entry animations.
